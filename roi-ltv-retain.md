## 广告术语

### 1. 转化率
1.1 存活率
```
激活量 / 安装量
```
1.2 转化率漏斗
```
拉新、促活、留存、付费转化四个步骤之间的每个步骤的转化率
```

### 2. 留存率
2.1 日留存率
```
第 N 日留存率 = 第 1 天新增且在第 N 天登录过的用户数 / 第 1 天新增用户数
```
2.2 周留存率
```
第 N 周留存率 = 第 1 周新增且在第 N 周登录过的用户数 / 第 1 周新增用户数
```
2.3 月留存率
```
第 N 周留存率 = 第 1 周新增且在第 N 周登录过的用户数 / 第 1 周新增用户数
```
2.4 加权留存率
```
各个时间段新增用户留存率的加权汇总
```

### 3. 用户付费指标
3.1 付费率 
```
付费人数 / 活跃人数, 也称付费转化率
```
3.2 ARPPU
```
(Average Revenue Per Paying User) 付费金额 / 付费人数, 每付费用户的平均收入
```
3.3 ARPU
```
(Average Revenue Per User) 付费金额 / 活跃人数, 每用户平均收入
```

### 4. 导入用户成本
4.1 CPC
```
(Cost Per Click) 广告投入总额 / 所投的广告带来的点击用户数, 单个点击用户的成本
```
4.2 CPA
```
(Cost Per Action) 广告投入总额 / 所投的广告带来的激活用户数, 单个激活用户的成本
```
4.3 CPR
```
(Cost Per Response) 广告投入总额 / 所投的广告带来的注册用户数, 单个注册用户的成本
```
4.4 CPL
```
(Cost Per Lead) 广告投入总额 / 所投的广告带来的新登录用户数, 单个登录用户的成本
```
4.5 CPS
```
(Cost Per Sale) 每次成交收费
```
4.6 CPM
```
(Cost Per Mille) 每千次展示收费
```

### 5. LTV
```
(Life Time Value) 生命周期总价值
每个用户平均的 LTV = 每月 ARPU x 用户按月计的平均生命周期
```

### 6. ROI
```
(Return on Investment) 投资回报率
ROI = 利润 / 投资总额 x 100%, 投资回报率, ROI > 0 则盈利, ROI < 0 则亏损
```
